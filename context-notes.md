# Context Notes

- 사용자는 1년 뒤 주가 예측이 부정확해 보인다고 했고, 이전에 제안한 개선 항목 전부를 최대한 구현해 달라고 요청했다.
- 핵심 판단은 시뮬레이션 횟수보다 입력 모형 검증이 중요하다는 것이다. 따라서 우선순위는 백테스트, 기준 모델 비교, 대시보드의 과신 표현 완화다.
- 대시보드 기본 1천만 경로는 몬테카를로 오차보다 모델 오차가 큰 상황에서 과하므로 300만 경로로 낮추고, 절약한 계산 여지를 검증/시나리오에 쓰는 방향으로 정한다.
- 모든 종목에 같은 점프 파라미터를 쓰는 문제를 줄이기 위해 대시보드 점프 모드를 고정 보수값, 역사적 추정값, 순수 GBM 중 선택하도록 한다.
- Kelly는 모델 입력 오차에 취약하므로 UI와 문서에서 권장 비중이 아니라 모형상 참고 비중으로 표현한다.
- 백테스트에는 Stellar 모형 외에 시장 prior, beta prior, flat median 기준 모델을 넣어 단일 모델 결과만 보는 문제를 줄인다.
- 대시보드 예측 분포에는 연율 μ 추정 불확실성을 경로별로 추가해 단일 표본 drift가 만드는 좁은 분포를 완화한다.
